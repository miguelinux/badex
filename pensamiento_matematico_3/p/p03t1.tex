\question ¿Cuántas veces debe intersectar una línea vertical a una grafica
          para que esta represente una función?

  \begin{oneparchoices}
    \choice cero veces
    \CorrectChoice una vez
    \choice dos veces
    \choice tres veces
  \end{oneparchoices}
